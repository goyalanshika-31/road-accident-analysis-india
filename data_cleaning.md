# Data Cleaning Process

## Objective
Prepare raw road accident dataset for analysis and dashboard creation.

## Cleaning Steps Performed

- Checked dataset structure and column names
- Removed missing or null values
- Standardized state names
- Converted data types where required
- Removed duplicate records
- Verified accident severity categories
- Ensured consistency in weather and road condition labels

## Tools Used
- Python (Pandas)
- NumPy
