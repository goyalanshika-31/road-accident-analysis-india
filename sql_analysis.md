# SQL Analysis Documentation

## Objective
Use SQL queries to extract meaningful insights from road accident data.

---

## 1. Total Accidents
**Purpose:**  
Calculate overall accident count.

**Method:**  
Used COUNT() aggregation function.

---

## 2. Accidents by State
**Purpose:**  
Identify accident-prone regions.

**Method:**  
Grouped records using GROUP BY state_name.

---

## 3. Accidents by Weather Conditions
**Purpose:**  
Understand environmental impact on accidents.

**Method:**  
Grouped data by weather_conditions.

---

## 4. Accident Severity Analysis
**Purpose:**  
Analyze distribution of accident severity levels.

**Method:**  
Aggregated records by severity category.
